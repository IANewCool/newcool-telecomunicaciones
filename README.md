# NewCool Telecomunicaciones - SUBTEL Chile

Portal educativo sobre telecomunicaciones y SUBTEL en Chile.

## Demo

[https://telecomunicaciones.newcool.io](https://telecomunicaciones.newcool.io)

## Caracteristicas

- Buscador de oficinas SUBTEL
- Comparador de operadores
- Calculadora de costo por GB
- Sistema de reclamos
- Cobertura por zona

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
